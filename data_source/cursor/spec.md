# Project Specification: Web Dashboard

## Overview
A modern web application built for data management and real-time visualization. The system follows a client-server architecture to ensure scalability and separation of concerns.

## Tech Stack
* **Frontend:** React (Vite) with Functional Components and Hooks.
* **Backend:** Node.js with Express.js framework.
* **Database:** PostgreSQL for relational data storage.
* **Authentication:** JWT-based stateless authentication.
* **API:** RESTful endpoints with JSON payloads.

## Architecture
1.  **Client Layer:** Handles UI rendering and state management.
2.  **API Layer:** Manages business logic, validation, and routing.
3.  **Data Layer:** Persistent storage using a relational schema.
